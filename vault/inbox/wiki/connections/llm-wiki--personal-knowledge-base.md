# LLM Wiki ↔ Personal Knowledge Base

## Relationship: 패턴과 목적

LLM Wiki는 Personal Knowledge Base(PKB)를 구축하기 위한 구체적인 패턴이다. PKB는 "무엇을 만들 것인가"이고, LLM Wiki는 "어떻게 만들고 유지할 것인가"이다.

## How LLM Wiki Solves PKB's Core Problem

PKB의 역사적 병목은 유지 관리 부담이다:
- 제텔카스텐 → 수동 카드 연결, 규모 한계
- Obsidian/Notion → 사용자 직접 작성/연결, 시간 소모
- RAG → 축적 없음, 매번 재검색

LLM Wiki는 LLM에게 유지 관리를 위임하여 이 병목을 제거한다. 인간은 소싱과 질문에 집중, LLM은 요약·교차 참조·정리·기록 관리를 담당.
