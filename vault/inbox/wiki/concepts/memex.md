# Memex

## Definition

Vannevar Bush가 1945년 에세이 "As We May Think"에서 제안한 가상의 개인 지식 장치. 문서, 기록, 통신을 저장하고, **연상적 경로(associative trails)**로 문서 간 연결을 만들어 탐색하는 개인 큐레이션 지식 저장소. 하이퍼텍스트와 월드와이드웹의 개념적 선구자로 평가받지만, Bush의 원래 비전은 웹보다 개인 지식 기반(PKB)에 더 가까웠다.

## Key Points

- **사적(Private)**: 공개 웹이 아닌 개인 소유의 지식 저장소
- **능동적 큐레이션**: 사용자가 직접 경로를 만들고 관리
- **연결의 가치**: 문서 간 연결이 문서 자체만큼 가치 있음
- **미해결 문제**: "누가 유지 관리를 하느냐" — Bush가 풀지 못한 핵심 과제

## LLM as the Missing Piece

Karpathy는 LLM 위키 패턴이 Memex의 정신적 후속이라고 주장한다. Memex가 구상한 모든 것(사적, 큐레이션, 연상적 경로)이 실현 가능하되, Bush가 해결하지 못한 유지 관리 문제를 LLM이 해결한다:

- 교차 참조 자동 생성/갱신
- 새 소스와 기존 지식의 자동 통합
- 모순 탐지 및 표시
- 10-15개 페이지 동시 갱신

## Historical Significance

1945년의 Memex → 1960년대 하이퍼텍스트(Ted Nelson) → 1989년 월드와이드웹(Tim Berners-Lee) → 2020년대 LLM 위키. 80년에 걸친 개인 지식 관리 비전의 점진적 실현.
